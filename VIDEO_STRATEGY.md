# 长文转短视频：核心差异与改造逻辑

将 Dan Koe 风格的长文转化为短视频（抖音/视频号/小红书）脚本，不能只是简单的“朗读”，必须进行结构性重组。

## 1. 核心差异对比

| 维度 | 长文逻辑 (阅读) | 短视频逻辑 (视听) |
| :--- | :--- | :--- |
| **开头** | 入坑引导，建立共鸣 | **黄金 3 秒**，抛出冲突或反直觉结论 |
| **节奏** | 娓娓道来，逻辑严密 | **高频切点**，每 5-8 秒必须有一个新信息点 |
| **语言** | 书面口语化，长短句结合 | **纯口语**，大白话，多用短句，甚至是不完整句 |
| **结尾** | 反思 + 行动建议 | **强引导**，点赞、关注、评论区见，或引导至私域 |
| **视觉** | 靠文字想象 | 靠画面（口播/混剪/文字动效）配合听觉 |

## 2. 改造三部曲

### 第一步：重写开头 (The Hook)
- **长文**：你有没有过这样的经历？刚开始写东西...
- **短视频**：别再幻想三个月就能拿结果了，这事儿你得干十年。

### 第二步：精简正文 (The Meat)
- 删掉所有铺垫词。
- 故事只讲最核心的冲突点。
- 金句要短，要能直接截图发朋友圈。

### 第三步：强化引导 (The CTA)
- 视频结尾不要只是“今日行动”，要变成“如果你也想通了，评论区打个‘坚持’，我拉你进群”。
